Welcome to Mimicry Framework

I really like front end development, CSS is something increanible. I really love bootstrap and Font Awesome, I think this is great projects, and then it inspired me to try to develop something. So I started Mimicry Framework, the idea is to create a grid sistem based on 24 columns to give to developer more control about the space and design, elements based on templates. You'll need two css files to use mimicry a mimicry.css base file and mimicry-theme.css file to change your components design when you want. The trully idea on this backgroud is that you develop once and you can change your site design just changing the mimicry-theme.css file.

Design Templates

You could build your own template if you want, mimicry is not ready yet, but I intend to create at leas 5 template:

  • Default -> based on bootstrap
  • Futuristic -> Based on Google products like Youtube
  • Flat -> Based on flat design
  • Realistic -> Based on Apple's design
  • Metro -> Based on metro design

Getting Started

  • Download the Mimicry framework
  • Add mimicry.css and mimicry-default.css to your html file
  • Set
  • Use classes

Grid system is based on 24 columns and then you can use this as bootstrap but:
You can create divs for Large devices as lg (TVs and large screens), Mediun devices as md (Nootbooks), Small devices as sm (Tablets), Extra small devices as xs (Smartphones) and Nano devices as nd (Smartwatches)
So the column is col-(lg, md, sm, xs, nd)-(1...24) -> col-lg-24
If you want to add a column with a width to you left col-(lg, md, sm, xs, nd)-offset-(1...24) -> col-md-offset-4
Else you can put a column to your right col-(lg, md, sm, xs, nd)-offset-right-(1...24) -> col-md-offset-right-4
